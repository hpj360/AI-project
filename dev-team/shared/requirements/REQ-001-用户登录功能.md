# REQ-001: 用户登录功能

## 背景
系统需要提供用户登录功能，支持多种登录方式，确保账户安全。

## 用户故事
作为 系统用户
我希望 能够使用账号密码或第三方账号登录
以便 安全地访问系统功能

## 验收标准
- [ ] 用户可以使用用户名和密码登录
- [ ] 用户可以使用手机号和验证码登录
- [ ] 用户可以使用微信扫码登录
- [ ] 登录失败时显示明确的错误提示
- [ ] 连续登录失败5次后锁定账户15分钟
- [ ] 登录成功后跳转到首页
- [ ] 支持"记住我"功能，7天内免登录

## 业务规则
1. 用户名长度为4-20个字符，支持字母、数字、下划线
2. 密码长度为8-20个字符，必须包含字母和数字
3. 验证码有效期为5分钟
4. 同一设备同时只能有一个有效登录会话
